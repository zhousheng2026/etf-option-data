## 技术分析知识库

### 布林带 (Bollinger Bands) 理论

**创始人**: 约翰·布林格 (John Bollinger)，1980年代提出，著有《Bollinger on Bollinger Bands》

**构成公式**:
- 中轨 = N日简单移动平均线 (SMA)，默认 N=20
- 上轨 = 中轨 + K × 标准差，默认 K=2
- 下轨 = 中轨 - K × 标准差

**统计学基础** (正态分布):
- ±1σ 范围: 68.27% 的价格数据
- ±2σ 范围: 95.45% 的价格数据 ← 布林带的理论基础
- ±3σ 范围: 99.73% 的价格数据

**Sigma值计算**:
```
Sigma = (收盘价 - 中轨) / 标准差
```
- Sigma = 2.0: 价格正好位于上轨
- Sigma > 2.0: 价格突破上轨
- Sigma = 3.0: 价格位于中轨+3σ，突破上轨后又上行1σ

**重现期计算** (基于正态分布单边概率):
| Sigma | 理论概率 P(X>σ) | 理论重现期 |
|:---:|:---:|:---:|
| 2.0σ | 2.28% | 约2个月 |
| 2.5σ | 0.62% | 约8个月 |
| 3.0σ | 0.135% | 约3年 |
| 3.25σ | 0.058% | 约7年 |

**重要注意**: 金融市场存在"肥尾效应"，实际极端事件频率高于理论值，实际重现期通常短于理论值。

## 错误预防机制

### 期货合约日期检查原则

**问题**: 期货合约代码包含年份月份（如EC2604=2026年4月），容易混淆过期合约

**解决方案**:
1. **提取合约日期**: 从合约代码解析年份和月份
2. **过期检查**: 只保留当前日期之后的合约（考虑交割月）
3. **主力合约优先**: 优先查询主力合约，避免远月流动性问题

**检查代码模板**:
```python
from datetime import datetime

def is_valid_contract(symbol, current_date=None):
    """检查合约是否未过期"""
    if current_date is None:
        current_date = datetime.now()
    
    # 提取年份月份（假设格式为 XXYMM 或 XXYYMM）
    # 例如: EC2604 -> 2026年4月
    import re
    match = re.search(r'(\d{2})(\d{2})$', symbol)
    if not match:
        return True  # 无法解析，默认有效
    
    year = int(match.group(1))
    month = int(match.group(2))
    
    # 处理年份（假设20xx年）
    if year < 50:
        year += 2000
    else:
        year += 1900
    
    contract_date = datetime(year, month, 1)
    
    # 合约在交割月前有效（通常提前1个月）
    return contract_date > current_date

# 使用示例
symbols = ['EC2504', 'EC2604', 'EC2606']
valid_symbols = [s for s in symbols if is_valid_contract(s)]
# 结果: ['EC2604', 'EC2606'] (2025年4月已过期)
```

### Sigma计算错误 - 2026-03-03 深刻教训

**错误描述**: 计算期货Sigma事件时，Sigma值严重高估（欧运指数7.09 vs 正确3.61，原油5.50 vs 正确3.31）

**根本原因**: 
1. **使用了静态布林带** - 只用昨日及之前的数据计算标准差
2. **忽略了今日价格对波动率的影响** - 当今日价格暴涨时，标准差会显著增大
3. **布林上轨是动态变化的** - 上轨位置会随今日价格调整，不是固定值

**错误计算过程**:
```
错误方法（静态）:
  标准差 = 历史20日标准差（不含今日）
  上轨 = 历史均线 + 2 × 历史标准差
  Sigma = (今日价格 - 历史均线) / 历史标准差  ← 分母太小，Sigma被高估

正确方法（动态）:
  标准差 = 最新20日标准差（含今日价格）
  上轨 = 最新均线 + 2 × 最新标准差
  Sigma = (今日价格 - 最新均线) / 最新标准差  ← 分母随价格调整，Sigma准确
```

**实际案例对比**:
| 品种 | 静态Sigma（错误） | 动态Sigma（正确） | 偏差 |
|:---:|:---:|:---:|:---:|
| EC2604 欧运指数 | 7.09 | 3.61 | -49% |
| SC2604 原油 | 5.50 | 3.31 | -40% |

**正确计算步骤**:
1. 获取最近19日历史收盘价
2. 加入今日实时价格，组成20个数据点
3. 重新计算20日均线和标准差（包含今日）
4. 计算布林上轨 = 新均线 + 2 × 新标准差
5. Sigma = (今日价格 - 新均线) / 新标准差

**检查清单（以后每次计算Sigma前）**:
- [ ] 标准差是否包含今日价格？
- [ ] 均线是否包含今日价格？
- [ ] 布林上轨是否基于最新数据计算？
- [ ] 如果今日价格大幅波动，是否重新计算统计参数？

**永远记住**: 布林带是**动态**的，不是静态的。当今日价格剧烈波动时，整个通道（中轨、上轨、标准差）都会随之调整。
