# 策略参数详解

## 资金设置

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 初始资金 | 100万-300万 | 低于100万难以分散，高于300万冲击成本大 |
| 单品种上限 | 50% | 避免单一指数黑天鹅 |
| 单笔仓位 | 30%-70% | 见入场条件分级 |

## 通道参数

| 参数 | 默认值 | 可调范围 |
|------|--------|----------|
| 通道K线数 | 20根 | 15-30根 |
| 最小触碰次数 | 2次 | 2-3次 |
| 平台斜率阈值 | 15° | 10°-20° |
| 波动范围阈值 | 2% | 1.5%-3% |

## 入场过滤

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 成交量倍数 | 1.3 | 突破时成交量/前20均量 |
| K线实体倍数 | 1.5 | 突破K线实体/前5平均实体 |
| IV Rank上限(多) | 55 | 避免高波动率入场 |
| IV Rank上限(空) | 65 | 做空放宽 |

## 离场参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 期权止损1 | 30% | 减仓50% |
| 期权止损2 | 50% | 清仓 |
| 通道内停留 | 3根K线 | 假突破确认 |
| 到期前平仓 | 3个交易日 | 30分钟周期更敏感 |

## 时间过滤

| 时段 | 操作 |
|------|------|
| 9:30-10:00 | 禁止开新仓 |
| 11:25-13:05 | 信号需二次确认 |
| 14:30-15:00 | 禁止开新仓 |

## 信号互斥

- 离场后等待: 3根K线
- 同时信号: 取成交量大者
