# 米筐(RiceQuant)版本策略使用说明

## 文件说明

| 文件 | 说明 |
|------|------|
| `ricequant_strategy.py` | 30分钟通道突破 - 双向版本（做多+做空） |
| `ricequant_long_only.py` | 30分钟通道突破 - 纯做多版本 |
| `ricequant_daily.py` | 日线20日高低点突破版本 |

## 使用方法

### 1. 在米筐平台创建策略

1. 登录 https://www.ricequant.com
2. 进入"策略研究" → "新建策略"
3. 将代码粘贴到编辑器
4. 点击"回测"运行

### 2. 修改标的

```python
# 默认使用沪深300ETF
context.symbol = '510300.XSHG'

# 可改为中证500ETF
context.symbol = '510500.XSHG'
```

### 3. 修改时间周期

```python
# 30分钟线（默认）
subscribe(context.symbol, '30m')

# 日线
# 删除subscribe行，使用默认日线
```

### 4. 修改回测时间

```python
config = {
    "base": {
        "start_date": "2023-01-01",  # 开始日期
        "end_date": "2025-02-25",    # 结束日期
        "accounts": {"stock": 1000000}  # 初始资金
    }
}
```

## 注意事项

1. **期权模拟**：米筐不直接支持期权回测，用ETF代替
2. **做空限制**：A股不支持直接做空，做空信号仅记录不执行
3. **滑点设置**：默认0.2%，可根据实际调整

## 回测结果查看

回测完成后可查看：
- 累计收益曲线
- 每日盈亏
- 交易记录
- 风险指标（夏普比率、最大回撤等）
