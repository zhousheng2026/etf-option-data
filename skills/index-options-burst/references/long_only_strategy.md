# 股指期权趋势爆发策略 - 纯做多版本

## 核心逻辑

**只做多，不做空**。买入看涨期权(Call)，捕捉上涨爆发行情。

**为什么只做多？**
- A股市场长期向上，做多胜率更高
- 国家队托底，下跌空间有限但上涨空间无限
- 做空时IV通常更高，成本增加
- 避免政策风险（限制做空、窗口指导等）

---

## 策略参数

| 参数 | 设置 |
|------|------|
| 时间周期 | 30分钟K线 |
| 标的指数 | 沪深300 / 中证500 |
| 合约类型 | 近月看涨期权(Call) |
| 行权价 | 虚值2-3档（Delta 0.25-0.35） |

---

## 入场信号

```
条件1: 收盘价突破通道上轨（或平台上沿）
条件2: 成交量 > 前20均量 × 1.3
条件3: K线实体 > 前5平均 × 1.5
条件4: IV Rank < 55

满足条件1+2 → 30%仓位
满足条件1+2+3 → 50%仓位
满足全部4个条件 → 70%仓位
```

---

## 离场规则

```
止损:
  - 期权跌30% → 减仓50%
  - 期权跌50% → 清仓

趋势跌破:
  - 跌破原通道上轨 → 减仓50%
  - 跌破通道中轨 → 清仓
  - 回到通道内3根K线 → 清仓
  - 到期前3个交易日 → 强制平仓
```

---

## 回测结果（模拟数据）

| 指标 | 纯做多版本 | 多空双向版本(做多部分) |
|------|-----------|----------------------|
| 交易次数 | 27 | 25 |
| **胜率** | **55.56%** | **56.00%** |
| **盈亏比** | **2.46** | **2.13** |
| 总收益 | +77.00% | - |
| 最大回撤 | 43.98% | - |
| 夏普比率 | 1.06 | - |

**对比结论**：纯做多版本盈亏比更高（2.46 vs 2.13），因为过滤了做空的亏损交易。

---

## 与双向版本对比

| 维度 | 纯做多 | 多空双向 |
|------|--------|----------|
| 胜率 | 55%+ | 多空平均51% |
| 盈亏比 | 2.46 | 1.59 |
| 回撤 | 较大（只做多，无对冲） | 较小（多空可对冲） |
| 适用市场 | 牛市/震荡向上 | 牛熊皆可 |
| 政策风险 | 低 | 做空有政策风险 |

---

## 使用建议

1. **牛市首选纯做多版本**，胜率更高、盈亏比更好
2. **震荡市或熊市**可考虑双向版本，但做空仓位要轻
3. **严格止损**，只做多时最大回撤可能较大
