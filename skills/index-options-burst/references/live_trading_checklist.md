### 2. 策略参数确认
```
时间周期: 30分钟K线
通道K线数: 20根（约5个交易日）
入场条件:
  - 突破通道上轨/跌破下轨
  - 成交量 > 前20均量 × 1.3
  - K线实体 > 前5平均 × 1.5
  - IV Rank < 55(多)/65(空)

离场条件:
  - 期权跌30% → 减仓50%
  - 期权跌50% → 清仓
  - 跌破通道上轨 → 减仓50%
  - 跌破通道中轨 → 清仓
  - 回到通道内3根K线 → 清仓

时间过滤:
  - 9:30-10:00 不入场
  - 14:30-15:00 不开新仓
```
