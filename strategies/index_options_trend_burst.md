# 股指期权趋势爆发策略（30分钟通道版）

## 核心逻辑

**目标**: 基于30分钟K线绘制趋势通道，通道/平台突破入场，通道跌破离场，捕捉股指期权爆发式行情。

---

## 一、时间周期与标的

| 维度 | 设置 |
|------|------|
| 分析周期 | 30分钟K线 |
| 标的指数 | 沪深300指数 (000300) / 中证500 (000905) |
| 合约类型 | 看涨期权(Call) / 认沽期权(Put) |
| 到期月份 | 近月合约 |
| 行权价 | 虚值2-3档（Delta ±0.25至±0.35） |

---

## 二、趋势通道绘制规则

### 上涨通道（做多用）

```
1. 取最近20根30分钟K线（约5个交易日）
2. 上轨: 连接两个明显的高点，要求第二个高点高于第一个
3. 下轨: 连接两个明显的低点，与上轨平行
4. 有效通道: 价格至少触碰上轨2次、下轨2次
5. 通道斜率: 
   - 斜率 > 0 为有效上涨通道
   - 斜率 < 15° 视为横盘平台，按平台突破处理
```

### 下跌通道（做空用）

```
1. 取最近20根30分钟K线
2. 下轨: 连接两个明显的低点，要求第二个低点低于第一个
3. 上轨: 连接两个明显的高点，与下轨平行
4. 有效通道: 价格至少触碰上轨2次、下轨2次
5. 通道斜率:
   - 斜率 < 0 为有效下跌通道
   - 斜率 > -15° 视为横盘平台，按平台跌破处理
```

### 平台整理（无方向）

```
1. 最近20根K线高低点波动范围 < 2%
2. 上轨 = 近期高点连线（水平或接近水平）
3. 下轨 = 近期低点连线（水平或接近水平）
4. 突破上轨 → 做多
5. 跌破下轨 → 做空
```

---

## 三、入场信号（30分钟通道突破）

### 做多入场

```
条件1: 价格收盘价突破通道上轨（或平台上沿）
条件2: 突破时成交量 > 前20根K线均量 × 1.3
条件3: 突破K线实体 > 前5根K线平均实体 × 1.5（拒绝小阳线突破）
条件4: IV Rank < 55

满足条件1+2 → 30%仓位
满足条件1+2+3 → 50%仓位
满足全部4个条件 → 70%仓位
```

### 做空入场

```
条件1: 价格收盘价跌破通道下轨（或平台下沿）
条件2: 跌破时成交量 > 前20根K线均量 × 1.3
条件3: 跌破K线实体 > 前5根K线平均实体 × 1.5
条件4: IV Rank < 65

满足条件1+2 → 30%仓位
满足条件1+2+3 → 50%仓位
满足全部4个条件 → 70%仓位
```

---

## 四、离场规则（通道跌破）

### 止损离场

```
1. 期权价格下跌30% → 止损50%仓位
2. 期权价格下跌50% → 清仓
```

### 趋势通道跌破离场（核心）

**做多离场**:
```
1. 价格收盘价跌破原通道上轨 → 减仓50%
2. 价格收盘价跌破通道中轨（或原下轨）→ 清仓
3. 价格重新跌回通道内部且停留3根K线 → 清仓
4. 到期前3个交易日 → 强制平仓（30分钟周期更敏感）
```

**做空离场**:
```
1. 价格收盘价突破原通道下轨 → 减仓50%
2. 价格收盘价突破通道中轨（或原上轨）→ 清仓
3. 价格重新涨回通道内部且停留3根K线 → 清仓
4. 到期前3个交易日 → 强制平仓
```

### 动态通道调整

```
持仓期间每收盘重新绘制通道:
- 若趋势延续，通道斜率加大 → 按新通道持有
- 若通道走平或反向 → 触发离场条件2
- 新通道与旧通道偏离 > 2% → 重新评估趋势有效性
```

## 五、信号互斥与过滤

### 多空互斥规则

```
1. 持有多头时:
   - 收到空头信号 → 忽略，维持多头
   - 多头离场后，需等待3根K线才能开空仓

2. 持有空头时:
   - 收到多头信号 → 忽略，维持空头
   - 空头离场后，需等待3根K线才能开多仓

3. 空仓时:
   - 同时出现多空信号 → 取成交量放大更明显的一方
   - 成交量相近 → 取先出现的信号
```

### 日内过滤

```
- 开盘前30分钟（9:30-10:00）不入场，避免隔夜跳空干扰
- 收盘前30分钟（14:30-15:00）不开新仓，避免尾盘异动
- 中午休市前后（11:25-13:05）信号需额外确认
```

## 六、策略特点

| 特点 | 说明 |
|------|------|
| 双向捕捉 | 上涨买Call，下跌买Put，不预测方向，只跟随趋势 |
| 高盈亏比 | 小止损博大行情，单笔盈利可能5-10倍 |
| 低胜率 | 趋势假突破常见，胜率约30-40% |
| 爆发捕捉 | 虚值期权在趋势加速期Gamma效应显著 |
| 严格纪律 | 趋势跌破即走，不扛单，多空不叠加 |

## 七、回测参数建议

- **回测区间**: 2020年至今（包含多轮牛熊）
- **K线数据**: 30分钟级别
- **手续费**: 开仓10元/张，平仓10元/张
- **滑点**: 2个最小变动价位
- **初始资金**: 100万

## 九、风险提示

1. **时间价值损耗**: 虚值期权时间价值衰减快，横盘即亏损
2. **流动性风险**: 深度虚值合约可能流动性不足
3. **波动率风险**: IV下降会抵消方向性收益
4. **杠杆风险**: 期权杠杆可达10-50倍，严格止损是必须的
5. **假突破**: 30分钟周期假突破频率高于日线，需严格按规则过滤

---

## 十、做空（认沽）特别注意事项

1. **IV偏斜**: 下跌时IV通常比上涨时高，认沽期权成本可能更高
2. **跳空风险**: 下跌爆发常伴随跳空，入场时机更难把握
3. **政策干预**: A股下跌时政策托底频繁，趋势持续性可能弱于上涨
4. **建议**: 做空仓位可比做多保守20%，或提高入场门槛（需4个条件全满足）
